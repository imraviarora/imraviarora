# 📊 Background
**EPAM Systems** (LSEG client) — *Aug 2023 → now*
Led migration of Jenkins → GitLab CI for 240+ microservices. Built reusable pipeline templates, integrated Semgrep / SonarQube / Black Duck, migrated secrets to HashiCorp Vault, automated AMI refresh across 20+ apps on a 90-day cycle.

**Healofy** — *Jul 2022 → Jul 2023*
Cut infra costs 35% through right-sizing and savings plans. Full GCP → AWS migration including PostgreSQL. Built API latency monitoring with BigQuery → Redshift + Lambda + EventBridge.

**Vetifly (UAE)** — *Apr 2021 → Jun 2022*
Architected multi-AZ EKS platform for 500K+ users. Owned the full AWS stack end to end — CodeBuild, ECS/EKS/Fargate, Lambda, RDS, CloudFront, WAF, GuardDuty, the works.

**Team Computers** — *Dec 2018 → Mar 2021*
50+ server distributed infrastructure. Wrote Ansible playbooks that saved the delivery team 80% of setup time. Multi-cloud Terraform across AWS, GCP, Azure.
